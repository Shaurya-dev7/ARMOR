# TeamUp Project Standards

This directory contains coding standards and guidelines adopted by the **Antigravity** agent for the TeamUp project.

## Reference Repository

These guidelines are sourced from the [vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills) repository, which provides specialized skills for AI coding agents.

## Included Guidelines

1.  **[React Best Practices](./react-best-practices.md)**
    - Source: `react-best-practices` skill
    - Focus: Performance optimization for React and Next.js (waterfalls, bundle size, etc.)

2.  **[Web Design Guidelines](./web-design-guidelines.md)**
    - Source: `web-design-guidelines` skill (via [vercel-labs/web-interface-guidelines](https://github.com/vercel-labs/web-interface-guidelines))
    - Focus: Accessibility, UX, Forms, and Animation best practices.

## Usage

These documents serve as the source of truth for code reviews and refactoring tasks performed by the AI agent.
